Sysco Invoices (Manual Dates)

For Sysco Excel invoices where there is no date column, upload each invoice and type the order date manually. These data points will be included in the price history line plots alongside other vendor invoices.
Sysco Format: Column H = Product description, Column K = Unit price.

Tip: Add one row per Sysco invoice. Each row needs a file and a date.

US Foods Invoices (Manual Dates)

For US Foods Excel invoices where there is no date column, upload each invoice and type the order date manually. These data points will be included in the price history line plots alongside other vendor invoices.
US Foods Format: Column B = Product description, Column G = Unit price.

Tip: Add one row per US Foods invoice. Each row needs a file and a date.
